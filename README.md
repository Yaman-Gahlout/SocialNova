# 🚀 SocialNova — Modern Social Media Web Application

SocialNova is a full-stack social media platform built using the MERN stack. It provides real-time communication, content sharing, and interactive social features similar to Instagram and WhatsApp.

---

## 🌟 Features

### 🔐 Authentication

- Secure login & signup using JWT
- HTTP-only cookies for authentication
- Protected routes

### 👤 User Features

- Follow / Unfollow users
- Suggested users system
- Profile view & edit

### 📸 Posts

- Create, like, and comment on posts
- Real-time like & comment updates (Socket.IO)

### 🎬 Reels

- Upload and view reels
- Randomized (shuffled) feed for better engagement

### 💬 Real-time Chat

- One-to-one messaging
- Image sharing support
- Online/offline user status
- Mobile responsive chat UI

### 🔔 Notifications

- Real-time notifications for:
  - Likes
  - Comments
  - Follows

### 📱 Stories

- Upload stories
- Seen/unseen indicator (Instagram-style)
- Story viewer tracking

---

## 🧠 Tech Stack

### Frontend

- React.js

- React Router
- Axios
- Socket.IO Client

### Backend

- Node.js
- Express.js
- MongoDB (Mongoose)
